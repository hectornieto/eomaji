We are happy to announce that under EOMAJI project we have published a series of capacity capacity building materials on evapotranspiration modelling, irrigation accounting and crop yield mapping:

<!--more-->

* Evapotranspiration modelling using the Two Source Energy Balance model, which source code can be found at [CSIC's Gitlab](https://git.csic.es/tech4agro/courses/eomaji-eoafrica) but can also be run in the cloud via [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/git/https%3A%2F%2Fgit.csic.es%2Ftech4agro%2Fcourses%2Feomaji-eoafrica/HEAD)

* SenET framework implemented for the Copernicus Data Space Ecosystem (CDSE) openEO API which source code can be found at [DHI's GitHub](https://github.com/DHI/Sen-ET-OpenEO-toolbox) and can be run on the cloud within the CDSE openEO API.

* Crop yield mapping using Copernicus imagery, which source code can be found at [CSIC's Gitlab](https://git.csic.es/tech4agro/courses/crop-yield) but can also be run in the cloud via [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/git/https%3A%2F%2Fgit.csic.es%2Ftech4agro%2Fcourses%2Fcrop-yield/HEAD).
